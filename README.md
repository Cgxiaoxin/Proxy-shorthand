# 待办事项微信小程序

一个功能完整、界面美观的待办事项管理微信小程序，帮助用户高效管理日常任务。

## ✨ 功能特性

### 📝 核心功能
- **添加待办事项** - 支持标题、描述、优先级和截止日期
- **编辑待办事项** - 随时修改任务内容和属性
- **删除待办事项** - 安全删除不需要的任务
- **标记完成** - 一键标记任务为已完成状态
- **筛选查看** - 按状态筛选（全部/待完成/已完成）

### 🎨 用户体验
- **现代化UI设计** - 采用Material Design风格
- **流畅动画效果** - 丰富的交互动画
- **响应式布局** - 适配不同屏幕尺寸
- **直观操作** - 简单易用的操作界面

### 📊 数据管理
- **本地存储** - 数据自动保存到本地
- **实时统计** - 显示任务总数、完成数、待完成数
- **优先级管理** - 支持低、中、高三个优先级
- **截止日期** - 支持设置任务截止时间

### 🚀 便捷功能
- **快速模板** - 预设常用任务模板
- **字符计数** - 实时显示输入字符数
- **表单验证** - 确保数据完整性
- **操作确认** - 重要操作需要确认

## 📱 页面结构

```
pages/
├── index/          # 主页面 - 待办事项列表
├── add/            # 添加页面 - 创建新待办事项
└── detail/         # 详情页面 - 查看和编辑待办事项
```

## 🛠️ 技术栈

- **框架**: 微信小程序原生开发
- **样式**: WXSS (类似CSS)
- **逻辑**: JavaScript ES6+
- **存储**: 微信小程序本地存储API

## 📦 安装和部署

### 1. 环境准备
- 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号并获取AppID

### 2. 项目配置
1. 克隆或下载项目代码
2. 在微信开发者工具中导入项目
3. 修改 `project.config.json` 中的 `appid` 为你的小程序AppID
4. 在 `app.json` 中根据需要调整页面配置

### 3. 图标配置
在 `images/` 目录下添加以下图标文件：
- `todo.png` / `todo-active.png` - 待办页面图标
- `add.png` / `add-active.png` - 添加页面图标
- `empty.png` - 空状态图片

### 4. 编译运行
1. 在微信开发者工具中点击"编译"
2. 在模拟器中预览效果
3. 真机调试验证功能

## 🎯 使用指南

### 添加待办事项
1. 点击底部"添加"标签页
2. 填写任务标题（必填）
3. 可选填写详细描述
4. 选择优先级（低/中/高）
5. 设置截止日期（可选）
6. 点击"添加待办"完成创建

### 管理待办事项
1. **查看列表**: 在主页面查看所有待办事项
2. **筛选任务**: 使用筛选栏查看不同状态的任务
3. **标记完成**: 点击任务前的圆圈标记完成状态
4. **编辑任务**: 点击编辑图标进入详情页面
5. **删除任务**: 点击删除图标确认删除

### 快速添加
- 使用预设模板快速创建常用任务
- 点击模板卡片自动填充表单内容

## 🔧 自定义配置

### 修改主题色
在 `app.wxss` 中修改CSS变量：
```css
:root {
  --primary-color: #4A90E2;
  --success-color: #2ED573;
  --danger-color: #FF4757;
}
```

### 添加新模板
在 `pages/add/add.js` 中的 `templates` 数组添加新模板：
```javascript
{
  id: 7,
  title: '新模板标题',
  description: '新模板描述'
}
```

### 调整页面配置
在 `app.json` 中修改页面路由和导航栏配置。
